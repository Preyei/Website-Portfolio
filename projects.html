Open source bioinformatics tool that aligns and finds conserved regions among multiple sequences using fasta files or manual input.


The main purpose of RumiSeq is to detect conserved regions across various sequences in fasta files. I utilized biopython to employ a sorting algorithm that I used to extract information used to detect conserved regions.


I developed RumiSeq while working for a research group. It strengthened my ability to work and learn simultaneously. I developed the ability to work autonomously and work with an interdisciplinary team.

Open source platform that guides a user in designing their own bowtie visualization in order to perform a hazard analysis.


Bowties are essential for providing a visual representation of showing the links between hazards, threats and consequences. For this project I employed UI/UX principles to ensure the web app was user friendly.


The bowtie designer was developed in parallel with working with the safety and security team to help achieve their goal of making safety tools accessible. The development of this project strengthened my critical thinking and design skills.
